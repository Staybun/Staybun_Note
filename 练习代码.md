# <font face="楷体" color=orange >Markdown基本语法</font>

1. **标题**
  # 一级标题
  ## 二级标题
  ### 三级标题
  #### 四级标题
  ##### 五级标题
  ###### 六级标题

2. **引用**
  > 编辑教程文档
  >> 引用的引用
3. **列表**
  1. 无序列表
  - 列表1
  + 列表2
  * 列表3 
  2. 有序列表
  3. 嵌套
  4. Todolist

       -    [x] a

       -    [ ] b

4. **表格**
  |左对齐|居中|右对齐|
  |:-----|:----:|----:|
  |a|b|c|

5. **段落**
    - 换行? - 两个以上空格后回车    
    - 分割线 - 三个* 
    ***

    - 字体
        |字体|代码|
    	|:--:|:--:|
    	|*斜体*|* * |
    	| ==高亮== | == == | 
    	|**粗体**|** **|
    	|***斜粗体***|*** ***|
    	|~~删除~~|~~ ~~| 

    - 脚注 
    多多支持[^1]!!!
6. **代码**
  ```c
  #include <stdio.h>
  int main()
  {
  	printf("helloworld");
  }
  ```
  若要在一行内引用：只需要反引号即`Hello`world

7. **超链接**
    >[链接名称](链接地址)
    <链接地址>
    [这是小白的主页](https://blog.csdn.net/qq_40818172?type=lately)
    or
    <https://blog.csdn.net/qq_40818172?type=lately>   

    [b站](https://www.bilibili.com/)

8. **图片**

    >![图片描述，可写可不写，但是中括号要有](图片地址，本地链接或者URL地址。)
    比如此文章的图片：
    ![卷不动的小白](https://i-blog.csdnimg.cn/blog_migrate/ec1c9a9c57e4d6494b5b36554f7af692.png)

10. **对于Markdown中的语法符号，前面家反斜线\即可以显示符号本身**

    >\\
    >\*
    >\_

     `$$`表示整行公式

     >$$E=mc^2$$
